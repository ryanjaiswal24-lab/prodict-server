Prodict - Personal Productivity Forecaster
-----------------------------------------
Backend: server (Node.js + Express + MongoDB)
Frontend: client (React + Vite + Tailwind + Chart.js)

Run backend:
  cd server
  npm install
  copy .env.example to .env and set MONGO_URI and JWT_SECRET
  npm run dev

Run frontend:
  cd client
  npm install
  npm run dev
  open http://localhost:3000
